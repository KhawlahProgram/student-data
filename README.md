# Student Salary Data Project

## 📌 Description
This is a simple Python project using pandas to create and display a dataset of students including their IDs, names, majors, and salaries.

## 🚀 Features
- Create a structured dataset using pandas
- Display student information in a DataFrame
- Simple and beginner-friendly Python project

## 🛠️ Technologies Used
- Python
- pandas
- numpy
